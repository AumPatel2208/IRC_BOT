## Commands

- CMD_EXIT = "exit";
  - bot exits the server
  - if the bot is inside of game, it will exit game.
- CMD_HELLO = "hello";
  - will reply hello
- CMD_HACK = "hack";
  - will fake hack channel and display the active users
- CMD_ATTACK = "attack" <name>;
  - banish the named person
- CMD_PLAY = "play";
  - play an adventure game.
